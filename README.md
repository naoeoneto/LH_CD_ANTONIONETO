# TESTE TÉCNICO LIGHTHOUSE - INDICIUM

**Para rodar esse projeto em seu computador, é necessário seguir os seguintes passos:**

1. Crie um ambiente virtual

```
python -m venv venv
```

2. Ative seu venv:

```bash
# Linux:
source venv/bin/activate

# Windows (Powershell):
.\venv\Scripts\activate

# Windows (Git Bash):
source venv/Scripts/activate
```

3. Instale o pacote listadas em requirements.txt:

```shell
pip install -r requirements.txt
```

4. Executar o arquivo **init**.py:

```shell
python analitics/__init__.py
```

Ou clique no botão de 'play' na barra superior. Lembrando que ele só estará disponível quando o arquivo citado estiver aberto.
